Функциональные возможности

Контроль выполнения программы
Выполнение пациентом упражнений по назначенной программе с автоматической фиксацией регулярности и соблюдения режима занятий
Количественный анализ движений
Автоматический подсчёт повторений и оценка параметров движения, включая анализ амплитуды в суставах
Оценка техники выполнения
Качественный анализ упражнений с предоставлением визуальной обратной связи пользователю
Сбор субъективной обратной связи
Получение данных о самочувствии пациента через встроенные опросы
Дистанционная коммуникация
Встроенный чат для взаимодействия пациента со специалистом
Аналитика и визуализация прогресса
Формирование статистики и наглядных графиков динамики показателей пациента во времени
Скрининговая диагностика
Проведение тестов для оценки готовности к переходу на следующий этап реабилитации
Кабинет специалиста и хранение данных
Веб-интерфейс для мониторинга выполнения программы, анализа динамики, корректировки плана реабилитации и хранения данных пациента
