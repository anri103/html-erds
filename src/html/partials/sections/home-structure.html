Структура программного комплекса

Мобильное приложение (Android) — предназначено для выполнения пациентом назначенной программы и передачи данных в систему
Серверная часть (Laravel) — обеспечивает хранение данных, обработку информации и взаимодействие компонентов системы через API
Веб-интерфейс — включает личный кабинет врача и административную панель для управления программами и пользователями
Обработка видеопотока производится на устройстве пользователя
Хранение данных осуществляется на серверах, размещённых на территории Российской Федерации
